# 员工请假管理办法

## 第一章 总则

### 第一条 目的

为规范公司员工请假管理流程，保障员工合法休假权益，确保公司各项工作正常运转，特制定本办法。

### 第二条 适用范围

本办法适用于公司全体在职员工，包括试用期员工。

## 第二章 假期类型

### 第三条 年假

- 员工累计工作满1年不满10年的，年休假5天
- 已满10年不满20年的，年休假10天
- 已满20年的，年休假15天
- 年假须在当年12月31日前休完，特殊情况经审批可顺延至次年3月31日
- 年假可一次性休完，也可分次休假，但每次不少于半天

### 第四条 事假

- 员工因个人事务需请事假的，须提前1个工作日申请
- 事假期间不发放工资
- 连续事假超过5天的，须提供相关证明材料
- 每月事假累计不得超过10天，每年事假累计不得超过30天

### 第五条 病假

- 员工因病需要治疗或休养的，可申请病假
- 病假须提供正规医院出具的诊断证明和病假条
- 病假1-3天的，由部门负责人审批
- 病假3天以上的，须经人力资源部审批
- 病假期间工资按以下标准发放：
  - 工龄不满2年：按工资的60%发放
  - 工龄2年以上不满4年：按工资的70%发放
  - 工龄4年以上不满6年：按工资的80%发放
  - 工龄6年以上：按工资的90%发放

### 第六条 婚假

- 员工本人结婚，可享受3天婚假
- 符合晚婚条件的（男25周岁、女23周岁），增加7天婚假，共10天
- 婚假须在领取结婚证后6个月内一次性休完
- 婚假期间工资照常发放

### 第七条 产假

- 女员工生育享受98天产假，其中产前可休15天
- 难产的增加产假15天
- 生育多胞胎的，每多生育1个婴儿增加15天产假
- 男员工配偶生育，可享受15天陪产假
- 产假、陪产假期间工资照常发放

### 第八条 丧假

- 员工直系亲属（父母、配偶、子女）去世的，可享受3天丧假
- 员工祖父母、外祖父母去世的，可享受1天丧假
- 需要到外地办理丧事的，可另给予路程假，路程假天数根据实际情况确定
- 丧假期间工资照常发放

## 第三章 请假流程

### 第九条 请假申请

1. 员工请假须通过OA系统提交请假申请
2. 申请内容包括：假期类型、起止时间、请假原因、工作交接安排
3. 需要附件材料的须一并上传（如病假条、结婚证等）

### 第十条 审批权限

| 请假天数 | 审批人 |
|---------|--------|
| 1天以内 | 直属主管 |
| 1-3天 | 部门负责人 |
| 3天以上 | 部门负责人 + 人力资源部 |
| 5天以上 | 部门负责人 + 人力资源部 + 分管副总 |

### 第十一条 紧急请假

遇紧急情况无法提前申请的，可先电话向直属主管报备，事后在1个工作日内补办请假手续。

## 第四章 违规处理

### 第十二条 违规情形

- 未经审批擅自休假的，按旷工处理
- 提供虚假请假材料的，给予警告处分，并取消相应假期
- 请假期满未续假或续假未获批准而逾期不到岗的，超过部分按旷工处理

## 第五章 附则

### 第十三条

本办法自2024年1月1日起执行，如有未尽事宜，由人力资源部负责解释。
